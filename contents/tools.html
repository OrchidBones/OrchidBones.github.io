工具


部分网址需要科学上网才能打开。



  • CharaConverter - 行走图转换器
  • Database Converter - 将数据库导入Excel或从Excel读取数据库
  • ExtractPatch - 提取补丁增量更新小工具
  • MV地图对白文本提取器
  • Iconset Wizard - 图标合成器
  • MVTextEditor - RMMV对话编辑器
  • MTool翻译器 - RPG Maker / Wolf RPG Editor 游戏翻译器
  • PKD_Protector - 插件脚本保护器 (仅电脑端有效)
  • RMMV Extended Generator - 额外的角色生成器 备用链接
  • RMMV Toybox (Mega网盘) - JSON编辑器、音频编辑器、存档编辑器、自动翻译器等
  • RMMV Resource Manager - RMMV资源管理器
  • RPG Maker Tileset Builder - 图块创建工具
  • Tileset-VA、XP素材转MV/MZ（高级版）
  • Tiny Project - 缩小工程体积，为图片增添像素抖动效果
  • TKool Character Animation Simulator - 行走图动画模拟器
  • Tkool Facetile Maker2 - 脸图编辑器，支持批量从立绘导出脸图
  • Tkool Image Tile Editor - 脸图组织器
  • Translator++ 及其 汉化版 - 强大的RPG Maker游戏翻译器，支持人工翻译和百度/谷歌自动翻译
  • New Resource Manager | Transparent Filter Extension for MV - 资源管理器，搭配RS_TransperantFilter.js可过滤行走图背景色
  • 将地图事件文本提取到Excel中便于翻译
  • Yanfly Icon Generator - YEP图标生成器
